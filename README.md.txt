# 🚀 HVAC Duct Cleaning Robot

## 🔍 Overview

The **HVAC Duct Cleaning Robot** is a semi-autonomous robotic system designed to navigate through air ducts and perform efficient dust detection and cleaning. This project aims to improve maintenance safety, reduce manual labor, and enhance air quality in HVAC systems.

---

## 🎯 Objective

* Develop a robot capable of **navigating confined duct spaces**
* Detect and remove dust using an **integrated cleaning mechanism**
* Reduce human intervention in hazardous and inaccessible environments

---

## ⚙️ System Architecture

### 🧠 Control System

* Microcontroller: STM32 / Arduino
* Optional Processing Unit: Raspberry Pi (for vision-based tasks)

### 🔌 Electrical System

* Power Supply: SMPS / Battery system
* Voltage Regulation: Buck converters
* Motor Drivers for DC and Servo control

### ⚡ Actuation

* High torque **Johnson DC motors** for locomotion
* Servo motors for directional control and mechanism actuation

### 📡 Sensors & Perception

* Time-of-Flight (ToF) sensor for distance measurement
* Camera module for inspection and monitoring

---

## 🛠️ Mechanical Design

* Compact chassis designed to fit inside HVAC ducts
* Multi-wheel drive system for stable movement
* Cleaning module integrated with a **vacuum mechanism**
* Load handling optimized using structural support and bearings

---

## 🔄 Working Principle

1. The robot is deployed inside the HVAC duct
2. It navigates using motor-driven wheels
3. Sensors detect obstacles and dust accumulation
4. The cleaning system activates to remove dust
5. Collected dust is handled via a vacuum mechanism
6. Optional camera feed provides real-time monitoring
